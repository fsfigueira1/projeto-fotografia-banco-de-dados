# 📸 V360 Fotografia

Sistema web completo para fotógrafos gerenciarem galerias privadas, vendas de fotos e atendimento aos clientes.

O projeto foi desenvolvido com foco em experiência do usuário, organização de eventos fotográficos, controle de acesso por galeria, carrinho de compras e fluxo de checkout.

## 🚀 Demonstração

🔗 Deploy: adicione aqui o link da Vercel  
🔗 Repositório: https://github.com/fsfigueira1/projeto-fotografia-banco-de-dados

## ✨ Funcionalidades

- Landing page moderna e responsiva
- Acesso do cliente por galeria privada
- Área administrativa para fotógrafo
- Cadastro, edição e exclusão de galerias
- Upload de fotos por evento
- Definição de preço por foto em cada galeria
- Carrinho de compras
- Cálculo de total no backend
- Integração com Stripe para checkout
- Autenticação com JWT
- Banco de dados MongoDB
- Validações no backend
- Interface responsiva
- Separação entre cliente e painel do fotógrafo
